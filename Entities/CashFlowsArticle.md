## Структура довідника
| Властивість | Тип даних | Примітки |
|---|---|---|
| Найменування </br> `Title` | Стрічка </br> `🔧` `String` | Відображувана назва. Складається із значка актуальності (якщо заблоковано), Коду та назви. Наприклад `⛔ PRD0205 • Корма`  |
| Актуальність </br> `Actuality` | [🎲 Актуальність](../Enums/EActuality.md) </br> `🔧` `EActuality` | Вираховується щоночі з полів `Актуально з` та `Актуально по`. Якщо поточна дата лежить між цими значеннями — актуально |
| Актуально з </br> `ActualFrom` | Дата час </br> `DateTime` |  |
| Актуально до </br> `ActualTill` | Дата час ❓ </br> `DateTime❓` |  |
| Код </br> `Code` | Стрічка </br> `String` |  |
| Назва </br> `Name` | Стрічка </br> `String` |  |
| Примітки </br> `Rem` | Стрічка </br> `String` |  |
